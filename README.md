# RemoteDesktop Server  
 
An RDP "setupper" app.
 
 
## Features 
  
- Allows RDP by one click  
- Check status and other informations     
- Should be used in pair with [RemoteDesktop 
Client](https://github.com/DominikTulak/RemoteDesktopClient.git)  
  
## Used resources 
 
- C#  
- XAML  
- .Net Framework 4
